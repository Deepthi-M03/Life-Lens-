**LifeLens**

**Description**

LifeLens is an AI-powered mobile application that uses facial recognition to detect a user’s age and dynamically controls app access. It ensures age-appropriate usage by displaying only suitable apps while restricting others, improving digital safety and personalized user experience.

**Features**
 -> Face detection and recognition
 -> Age estimation using AI
 -> Age-based app restriction system
 -> Child safety and parental control support
 -> Dynamic app display based on user age
**Tech Stack**
 -> React.js
 -> Kotlin (Android Development)
 -> AI/ML for face and age detection
**Future Scope**
 -> Improve accuracy of age detection
 -> Add advanced parental control settings
 -> Real-time monitoring and alerts
 -> Integration with device security systems
